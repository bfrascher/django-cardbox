Filter Reset

You can prefix any expression with a ~ to negate it.
By default all cards will be returned that contain the given expression. To override that behaviour you can use the following prefixes (following after the ~ if you use it):
  • =: equals/exact
  • <=: lesser equals
  • >=: greater equals
  • <: lesser
  • >: greater
If you use multiple expressions in the same filter only cards matching all expressions will be returned (with the exception of the set filter which will return all cards that are in any of the specified sets). You can also use & (and) and | (or) explicitly as well as group the expressions using ( and ).
If you have an expression you want to search for that is longer than a single word, surround it with " or '. The expression will be matched case sensitive. Using r" or r' as the opening quote will interpret the contained expression as a regular expression.
